🖥️ Computer Issues Quick Start Guide

Identify and solve hardware, software, and printer problems

HARDWARE RELATED ISSUES

🔊 POST BEEP CODES (AWARD BIOS)

  • 1 short beep – System boot successful.
  • 2 short beeps – CMOS settings error. Reset BIOS to default.
  • 1 long, 1 short – Memory or motherboard error. Reseat RAM, test sticks individually, replace faulty RAM or motherboard.
  • 1 long, 2 short – Monitor or graphics card error. Reseat GPU, reconnect monitor, replace GPU or monitor.
  • 1 long, 3 short – Keyboard error. Reconnect keyboard, try another port, replace keyboard.
  • 1 long, 9 short – BIOS ROM error. BIOS corrupted. Reflash BIOS.
  • Continuous long beeps – Graphics card not inserted properly. Reseat GPU.
  • Continuous short beeps – Power error. Reconnect components one by one, replace PSU if needed.

🔌 BOOT AND POWER ISSUES

🔌 Computer does not power on

Causes: faulty PSU, dead motherboard, loose power cable, faulty power button.

Fix: test wall outlet, replace cable, test PSU, jump motherboard power pins, replace PSU or motherboard.

🖥️ Computer powers on but no display

Causes: GPU not seated, RAM not seated, monitor cable loose, wrong input.

Fix: reseat GPU and RAM, check monitor cable, change monitor input, test with another monitor.

🔥 Computer turns on then immediately off

Causes: CPU cooler not attached, overheating, PSU failure.

Fix: reattach CPU cooler, replace thermal paste, test PSU.

🔄 Computer stuck in boot loop

Causes: corrupted BIOS, faulty RAM, failing drive.

Fix: reset BIOS, test RAM, disconnect drives, reinstall OS.

❄️ CPU AND COOLING ISSUES

🌡️ Overheating

Symptoms: shutdowns, thermal throttling, loud fans.

Causes: dust, poor airflow, failing fans, dried thermal paste.

Fix: clean dust, add fans, replace thermal paste, upgrade cooler.

❄️ CPU fan not spinning

Causes: dead fan, blocked blades, faulty header.

Fix: clean fan, reconnect cable, test on another header, replace fan.

📈 High CPU usage

Causes: background apps, malware, Windows indexing.

Fix: end tasks, disable startup apps, scan for malware.

🧠 RAM ISSUES

🧠 RAM Problems

Symptoms: random crashes, BSOD, freezing, failure to boot.

Causes: faulty RAM stick, incompatible RAM, dirty contacts.

Fix: test RAM sticks individually, clean contacts, replace faulty RAM.

🔍 Computer boots but only detects some RAM

Causes: bad slot, incompatible RAM.

Fix: test each slot, update BIOS, replace motherboard if slot is dead.

🔧 MOTHERBOARD ISSUES

🔧 Motherboard Problems

Symptoms: no power, random shutdowns, USB ports failing.

Causes: damaged capacitors, short circuits, BIOS corruption.

Fix: inspect for bulging capacitors, reset BIOS, replace motherboard.

🔌 USB ports not working

Causes: damaged ports, driver issues, motherboard failure.

Fix: reinstall USB drivers, test other ports, use powered USB hub, replace motherboard.

🎮 GRAPHICS CARD ISSUES

🎮 No display from GPU

Causes: GPU not seated, faulty cable, wrong output.

Fix: reseat GPU, try different cable, switch to another port.

👾 Artifacts on screen

Causes: overheating GPU, failing VRAM.

Fix: clean GPU, increase airflow, replace GPU.

💨 GPU fans not spinning

Causes: fan curve, dust, dead fan.

Fix: clean fans, adjust fan curve, replace fan.

⚡ POWER SUPPLY ISSUES

⚡ Power Supply Problems

Symptoms: random shutdowns, burning smell, system instability.

Causes: failing PSU, overloaded PSU, cheap power supply.

Fix: test with PSU tester, replace PSU with higher quality unit.

🔇 PSU fan not spinning

Causes: dust, dead fan.

Fix: replace PSU (do not open PSU).

💽 STORAGE ISSUES

💽 Hard drive clicking

Causes: mechanical failure.

Fix: back up data immediately, replace drive.

🐌 Hard drive slow

Causes: fragmentation, failing drive.

Fix: defragment HDD, run CHKDSK, replace drive.

💿 SSD not detected

Causes: loose cable, BIOS settings, dead SSD.

Fix: reconnect cable, enable AHCI, update BIOS, replace SSD.

📁 Corrupted files

Causes: sudden shutdowns, failing drive.

Fix: run CHKDSK, restore from backup, replace drive.

⌨️ PERIPHERAL ISSUES

⌨️ Keyboard not working

Causes: dead keyboard, USB port failure.

Fix: try another port, replace keyboard.

🖱️ Mouse lagging

Causes: low battery, interference, bad sensor.

Fix: replace batteries, use wired mode, clean sensor.

📺 Monitor flickering

Causes: loose cable, wrong refresh rate.

Fix: reconnect cable, change refresh rate, replace cable.

🔊 NOISE ISSUES

🔊 Loud fan noise

Causes: worn-out fans, dust, high RPM.

Fix: clean fans, replace fans, adjust fan curve.

⚙️ Grinding noise

Causes: failing HDD or fan.

Fix: replace HDD, replace fan.

🪛 Vibration noise

Causes: loose screws, unbalanced fans.

Fix: tighten screws, replace fan.

PREVENTATIVE MAINTENANCE

  • 🧹 Clean dust every 6 to 12 months.
  • 🔄 Update drivers regularly.
  • 💻 Install OS updates.
  • 🛡️ Run antivirus scans.
  • 💾 Back up data frequently.
  • 🌬️ Keep airflow unobstructed.
  • 🧴 Replace thermal paste every few years.
  • 🔋 Avoid cheap power supplies.
  • ⚡ Use surge protection.
  • 🔧 Keep software updated.
  • 🚫 Avoid installing unknown programs.
  • 🌡️ Monitor temperatures with hardware tools.

💻 SOFTWARE RELATED ISSUES

💀 Blue Screen of Death (BSOD)

Causes: corrupted system files, incorrect drivers, faulty RAM, overheating.

Fix: update drivers, run CHKDSK, test RAM, check temperatures.

⏸️ Application not responding

Causes: software bugs, low RAM, slow internet.

Fix: restart app, update software, reinstall app.

🐌 Slow performance

Causes: too many startup apps, low disk space, malware.

Fix: disable startup apps, uninstall unused programs, run disk cleanup.

🚫 Operating system won’t boot

Causes: corrupted boot files, failed update.

Fix: run startup repair, restore system, reinstall OS.

🔗 Driver conflicts

Symptoms: crashes, hardware not detected.

Fix: reinstall drivers, roll back updates.

🌐 NETWORK ISSUES

🌐 No internet

Causes: router issues, cable problems, DNS issues.

Fix: restart router, check cables, change DNS.

🐌 Slow internet

Causes: congestion, weak Wi-Fi signal.

Fix: move closer to router, use Ethernet, restart router.

📶 Wi-Fi disconnecting

Causes: interference, outdated drivers.

Fix: update Wi-Fi drivers, change Wi-Fi channel.

🔌 Ethernet not working

Causes: bad cable, disabled adapter.

Fix: replace cable, enable adapter, reinstall drivers.

🛡️ MALWARE ISSUES

🦠 Malware Symptoms

Symptoms: pop-ups, slow performance, unknown programs.

Fix: run antivirus scan, remove threats, reset browser, restore from backup.

💰 Ransomware

Symptoms: files encrypted, ransom note.

Fix: disconnect from network, restore from backup.

🕵️ Spyware

Symptoms: browser redirects, unknown extensions.

Fix: remove extensions, scan for malware.

🖨️ PRINTER ISSUES

🖨️ Printer not responding

Causes: spooler failure, driver issues.

Fix: restart spoolsv.exe, reinstall drivers.

📄 Paper jams

Causes: debris, worn rollers.

Fix: remove jammed paper, clean rollers.

🎨 Poor print quality

Causes: clogged nozzles, low ink.

Fix: clean print heads, replace ink.

❌ Printer offline

Causes: network issues.

Fix: reconnect printer, restart router.

📡 Wireless printer not connecting

Causes: weak signal, wrong password.

Fix: move printer closer, reconnect Wi-Fi.

🛠️ PREVENTATIVE MAINTENANCE

  • 🧹 Clean dust, vents, and fans every 3-6 months (industry best practice).
  • 🔁 Update OS and drivers monthly; check vendor notifications weekly.
  • 💾 Schedule weekly backups to local and cloud stores; test recovery quarterly.
  • 🛡️ Maintain anti-malware active, schedule daily quick scans and weekly full scans.
  • 🌡️ Monitor system temperatures; set fan curves and alerts for 70°C+ CPU/GPU.
  • 🔋 Inspect power cables and surge protector annually; replace if worn.
  • ⚙️ Run Disk Cleanup, CHKDSK, and SSD TRIM monthly.
  • 🧰 Track firmware updates for BIOS, firmware, and peripherals.
  • 🧴 Reapply thermal paste every 2-3 years, or immediately if temperatures rise abruptly.
  • 📦 Document change history and troubleshooting steps; keep a recovery USB and system image.
